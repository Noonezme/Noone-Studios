# Neon Shock (2D)

## "Survive the Glitch. Master the Grid."

Neon Shock is a cyberpunk survival shooter made for the browser.

You play as the Neon Interceptor and fight against endless waves of corrupted entities while trying to survive different environments and challenges.

## About

This project was built under **NoOne Studios** as an experiment to see how far browser games could be pushed.

The goal was to create a fast, visually interesting shooter with a custom game engine, smooth performance, and a lot of effects without relying on a traditional game engine.

## Features

- 5 different seasons/environments
- Multiple weapons and upgrades
- Boss fights
- Particle effects and dynamic lighting
- Custom 60FPS browser engine
- Cyberpunk / synthwave style

## Built With

- Custom game systems
- Web technologies
- Performance optimization
- UI and gameplay design
